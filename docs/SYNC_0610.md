## Phase 3 架构

```
用户 ←微信→ AstrBot（本地） ←HTTP→ FastAPI（云端）
         ↑                              ↓
   proactive_chat               watchdog_state
   30分钟沉默→主动关怀           45分钟→PushDeer
                                60分钟→短信告警
```

- echo_watchdog 插件：用户发消息 → POST /internal/heartbeat → 重置看门狗
- proactive_chat 插件：30 分钟沉默 → AstrBot 主动发微信关怀
- 两个插件配合：一个管"用户说话时确认安全"，一个管"用户不说话时主动问"
