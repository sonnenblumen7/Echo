## 踩坑

1. **app.json permission 格式**：`{"desc":"..."}` 错误，必须 `{"scope.userLocation":{"desc":"..."}}`
2. **bindlongtap 不是 3 秒**：微信约 350ms 触发。SOS 改用 touchstart + setTimeout 3 秒手动计时
3. **飞行模式测试不可行**：小程序被系统挂起，setInterval 停止。改用停后端服务模拟信号盲区
4. **开发者工具缓存**：新代码不生效需「清缓存 → 全部清除 → 编译」

## 架构决策

- SOS 触发后心跳继续（SOS 是一次性事件，不是永久状态，联系人已收到短信即可）
- 手动心跳不重置主定时器，只重置倒计时显示（避免干扰自动周期）
- 后端 heartbeat 路由新增坐标日志（之前只有 SOS 有日志）
