# 微信小程序提审材料

## 小程序名称

**Echo 防失联看门狗**

## 小程序简介

Echo 是一款面向户外自驾旅行者的防失联安全工具。用户开启守护后，小程序自动定时上报位置心跳。当系统检测到用户长时间未响应时，自动通过短信和推送通知紧急联系人，并附带最后已知位置和地图导航链接。适用于无人区、高原、信号薄弱等偏远地区的单人出行场景。

## 隐私相关说明

- 获取用户位置：用于定时上报心跳坐标，实现防失联监控
- 位置数据仅用于安全告警，不用于商业用途
- 紧急联系人信息仅存储在用户设备和安全服务器中
- 用户可随时在设置页删除联系人数据
